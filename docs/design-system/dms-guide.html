Document Management System - Style Guide

The DMS is a digital solution for the management of documents across Health Alliance products. It empowers internal users to handle the hosting of documents used on public facing platforms, and acts as the source of truth for the necessary metadata of all documents.

Internal Design Guide

The DMS will pull from the ha-internal stylesheet and JS file hosted on the Content-Haorg repository.

CSS

<link rel="stylesheet" type="text/css" href="./bundle/ha-internal.css">

JS

<script type="text/javascript" src="/js/ha-internal.js/"></script>

Activating DMS Specific Organisms

Placing "dms" class on the body tag of the page allows the usage of organisms specific to the DMS within the body.

<body class="dms"></body>
