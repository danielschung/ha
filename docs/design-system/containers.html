Containers

The containers in this design system cover most cases. Used in tandem with the auto-grid system and layout classes, you can achieve total control over the content within containers. Define a container by applying class container to a div.

Width Containers

Width containers are typically used as page wrappers. These containers apply a method to the design system to give control classes for the various widths you might want to achieve.

Full Container

To achieve a container that reaches the entire width of the parent element, apply class full to the container. To ensure it stretches the entire width of the browser, make sure it is a child of the body.

This container has full width.

<div class="full"></div>

Container Limit

Limited containers have a max-width of 1200px and are centered on the parent. To center it on the page, make sure it is a child of the body.

This container has limited width.

<div class="limit"></div>

Colored Container

In some cases you might want a simple container with a colored background. In these cases, just add class for the color you'd like to add. It will apply padding and border-radius styles as well.

This div is purple and has padding and border-radius styles applied.

<div class="purple-30">
  <p>This div is purple and has padding and border-radius styles applied.</p>
</div>

Closed Container

In other cases, you might want a container to have a card-like appearance without the interactivity of a card component. Closed containers apply the card border styles in a simple container. It is applied with class closed. The background-color of the closed container can also be set by adding a color class to the element.

This container has a card-like appearance.

<div class="closed">
  <p>This container has a card-like appearance.</p>
</div>

Divider Container

Divider containers are open aside from a gray border at the bottom. It can be used in cases where you need a single divider between listed components.

One Column Example

row 2

row 3

<div class="divider">
  <p>row 1</p>
</div>
<div class="divider">
  <p>row 2</p>
</div>
<div class="divider">
  <p>row 3</p>
</div>
