Buttons

Button styles can be achieved by placing a mix of classes on the button. Functionality of components requires ha-internal.js.

Purple 

<button class="purple">Default</button>

Clear 

<button class="clear">Default</button>

Small 

<button class="small">Default</button>

Small Buttons

The small class can be added to any button to achive a smaller button size.

Purple Small 

<button class="purple small">Default</button>

Clear Small 

<button class="clear small">Default</button>

Small Success 

<button class="small success">Default</button>

Round Buttons

Clear Round 

<button class="clear round">Default</button>

Small Round 

<button class="small round">Default</button>

Purple Small Round

Clear Small Round

Text Buttons

Text buttons use the icomoon icon "arrow-right2" and can be inserted using an i tag with class "arrow-right2" inside the button.

<button class="text">Default<i class="icon-arrow-right2"></i></button>

Circle Button

The Circle button displays alternative options on click. It requires a white icon, which can be achieved by adding a class "white" on the icon. On click, the alternative options div will be displayed. This div must be the next sibling element after the circle button.

<button class="circle"></button>
<div> <!--must go right after circle button -->
<button class="clear round">Option 1</button>
<button class="clear round">Option 2</button>
</div>

Initialize

<script>
  include(circleButton());
</script>

</script>

Favorite Button

The favorite button can be applied with or without text. Clicking the component applies class active which makes it gold.

<button class="favorite"><div class="star"></div>Mark as Favorite</button>

Initialize

<script>
  include(favoriteButton());
</script>

Tabs

Tabs can be achieved by creating a div with class "tabs" that contains at least one div with class "tab-header" and a div with class "content." The "content" div contains the tab content and needs to be the last child.

Option 1
Option 2
Option 3
<div class="tabs">
  <div class="tab-header active>Option 1</div>
  <div class="tab-header>Option 2</div>
  <div class="tab-header>Option 3</div>
  <div class="content></div>
</div>
