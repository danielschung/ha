Design System

This is Health Alliance's design system for digital products to maintain a consistent brand and experience. This design-system mainly addresses internal products for now, but hopefully can eventually encompass all our products. It will allow designers to tightly establish brand guidelines and allow developers to deploy styles more quickly and accurately in development.

Getting Started

Our design system requires the use of one file and one file. Both resources are hosted on the Content-Haorg repository. Linking these up gives access to all the styles in this design system.

CSS

<link rel="stylesheet" type="text/css" href="./bundle/ha-internal.css">

JS

<script type="text/javascript" src="/js/ha-internal.js"></script>

Initialize Components

Components marked need to be initalized to work. To do this, simply call the function include() on the page and pass the component function. You can pass multiple component functions by separating them with a comma. Each component's function will be listed in its documention.

<script>
  include( circleButton(), optionsButton(), filterButton() );
</script>
