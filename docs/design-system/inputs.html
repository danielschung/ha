Inputs

Form inputs need to be placed inside a div with class "input." This is to ensure the label and input styles are applied correctly. Labels are linked to input by unique id.

Text Input 

<div class="input">
  <label for="text">Label</label>
  <input type="text" placeholder="Default" id="text">
</div>

File Input

Full Component

File inputs cannot be styled, but this component uses a workaround. The input itself is hidden, and styled HTML is injected into the div that comes after the input. This only works if the div with class file comes immediately after the input.

<div class="input">
  <label for="Default">Label</label>
  <input type="file" id="file">
  <div class="file"></div> <!-- placed immediately after input -->
</div>

Initialize

<script>
  include(fileInput());
</script>

Simple Version

If you are having trouble implementing the full component, here is a simple but less styled version for use. 

<div class="simple-input">
  <label for="simple-file">Label</label>
  <input type="file" id="simple-file">
</div>

Checkbox Input

CSS editing for checkboxes is limited, so these styles are implemented onto a div with class "checkbox." Labels are placed to the right of the checkbox, you can achieve this by placing the label after the input. The div with class "checkbox" must go immedately after the label to achieve correct styling.

<div class="input">
  <input type="checkbox" id="checkbox">
  <label for="checkbox">Label</label> <!-- label sits to the right -->
  <div class="checkbox"></div> <!-- placed immediately after label -->
</div>

Initialize

<script>
  include(checkboxRadioInput());
</script>

Radio Inputs

CSS editing for radio buttons is limited, so the radio input is hidden and the styles are implemented onto a div with class "radio." Labels are placed to the right of the radio button, you can achieve this by placing the label after the input. Radio buttons that share the same name will belong to the same group. The div with class "radio" must go immediately after the label to achieve correct styling.

<div class="input">
  <input type="radio" name="same" id="radio1">
  <label for="radio1">Group A</label> <!-- label sits to the right -->
  <div class="radio"></div> <!-- placed immediately after label -->
</div>
<div class="input">
  <input type="radio" name="same" id="radio2">
  <label for="radio2">Group A</label> <!-- label sits to the right -->
  <div class="radio"></div> <!-- placed immediately after label -->
</div>

Initialize

<script>
  include(checkboxRadioInput());
</script>

Toggle Input

The toggle input is an advanced checkbox input. Labels are placed to the right of the toggle, which can be achieved by placing it after the input. The styles are implemented onto the div with class "toggle." This div must go immediately after the label.

<div class="input">
  <input type="checkbox" id="toggle">
  <label for="toggle">Enable</label> <!-- label sits to the right -->
  <div class="toggle"></div> <!-- placed immediately after label -->
</div>

Initialize

<script>
  include(checkboxRadioInput());
</script>

Search Inputs

Search bars are text inputs with unique styling. The pieces that comprise the search are the search-icon, the text input, the clear button, and the go button which all belong inside the div with class "input." The order of these child components is important to maintain accurate styling. Clicking on the clear button will clear the text value of the input.

<div class="input"> <!-- child components must be placed in this order -->
  <div class="search-icon"></div>
  <input type="text" class="search" placeholder="Search">
  <div class="search-clear"></div>
  <div class="search-go"></div>
</div>

Initialize

<script>
  include(searchBar());
</script>

Number Inputs

Quantity Input

The quantity input allows user to increment a number value in an input with + and - controls. Label can be optionally removed, but button.minus and button.plus are needed. Order also matters.

<div class="input quantity">
  <label for="quantity-input">Label</label>
  <button class="minus"></button>
  <input type="number" id="quantity-input" placeholder="0" pattern="[0-9]{2}">
  <button class="plus"></button>
</div>

Initialize

<script>
  include(quantity());
</script>
