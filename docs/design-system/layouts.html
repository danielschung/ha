Auto Grid System

The grid system consists of auto-grids and cell. An auto-grid holds a number of child cell. The auto-grid reads the number of cell as child elements and automatically sets even spacing and width for all column divs. The auto grid works for up to 8 cell.

Alignment

By placing a class on a column div you can set the alignment behavior to right or center. cell have an alignment left as default.

<div class="auto-grid">
  <div class="column">
    <button class="clear">Left</button>
  </div>
  <div class="column center">
    <button class="clear">Center</button>
  </div>
  <div class="column right">
    <button class="clear">Right</button>
  </div>
</div>

Vertical Center

In some cases you might want an element to be vertically centered. You can achieve this by applying class vertical-center to the element. For this class to work, the height of the parent must be set.

This is vertically centered

<div class="full"> <!-- height is set to 200px -->
  <p class="vertical-center">This is vertically centered</p>
</div>
