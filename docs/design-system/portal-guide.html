Portal - Style Guide

The post-login portals are UI interfaces for users to access and order documents. There is a currently are currently four portals: one for Brokers, Employers, Providers, and Members.

Internal Design Guide

The portals will pull from the ha-internal stylesheet and JS file hosted on the Content-Haorg repository.

CSS

<link rel="stylesheet" type="text/css" href="./bundle/ha-internal.css">

JS

<script type="text/javascript" src="/js/ha-internal.js/"></script>

Activating Portal Specific Organisms

Placing "fr" class on the body tag of the page allows the usage of organisms specific to the F&R portals within the body.

<body class="fr"></body>
